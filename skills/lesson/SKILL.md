# 教訓の抽出と保存

トリガーされたら、直近の会話から教訓を抽出して長期記憶に保存する。

## 手順

1. **直近の会話をスキャン** — 話題になった落とし穴、バグ修正、重要な知見を特定
2. **技術レイヤーを保存**（category: fact, importance >= 0.8）:
   ```
   問題: [症状]。原因: [根本原因]。修正: [解決策]。予防: [回避方法]。
   ```
3. **原則レイヤーを保存**（category: decision, importance >= 0.85）:
   ```
   判断原則 [タグ]: [行動ルール]。トリガー: [いつ]。アクション: [何をする]。
   ```
4. **検証** — `memory_recall` でキーワード検索し、両エントリが取得できることを確認
5. **報告** — 何を保存したか簡潔に報告

## ルール

- 各エントリは短くアトミックに（500文字以内）
- 明確な教訓が見つからない場合は、何を保存すべきか質問する
- 既存の記憶と重複しないよう、保存前に `memory_recall` で確認する
